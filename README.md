# ✦ Lumen Arcana

> A quiet ritual for those who want to look inward.

A daily tarot reflection web app. Draw one Major Arcana card per day and receive a personal reflection generated by AI, combining the card's meaning with your daily horoscope.

**Live:** [lumenarcana.vercel.app](https://lumenarcana.vercel.app)

---

## Stack

- React + Vite + TypeScript
- Groq API (Llama 3.3) for AI reflections
- Vercel Edge Functions
- LocalStorage only

## Features

- Daily Major Arcana card draw
- AI-generated reflection based on card meaning + horoscope (APIs)
- Reading history + card collection
- Fully responsive
